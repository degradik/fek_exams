Базовые уравнения

Линейное: ax + b = 0

Перенесите свободное число в другую часть и разделите на коэффициент при x.

3x − 6 = 0 → 3x = 6 → x = 2

Квадратное: ax² + bx + c = 0

Вычислите дискриминант D = b² − 4ac. Если D > 0 — два корня, D = 0 — один, D < 0 — действительных корней нет.

x₁,₂ = (−b ± √D) / 2a

По Виету: x₁ + x₂ = −b/a,   x₁x₂ = c/a.

Модуль, дробь и корень

  • |A| = b при b ≥ 0 означает два случая: A = b или A = −b.
  • В дробном уравнении сначала выпишите ОДЗ: знаменатель не равен нулю.
  • При возведении в квадрат могут появиться посторонние корни — подставьте ответ в исходное уравнение.
Всегда проверяйте найденный x подстановкой: это быстрее, чем потерять баллы из-за знака.

Что значит «черточка» у f: f′

f′(x) — производная. Она показывает, как быстро меняется функция, или наклон графика в точке. Запись f′(2) означает: сначала найти производную, затем подставить x = 2.

(xⁿ)′ = nxⁿ⁻¹   ·   (C)′ = 0   ·   (eˣ)′ = eˣ   ·   (√x)′ = 1/(2√x)
  1. Берём производную каждого слагаемого.
  2. Упрощаем выражение.
  3. Подставляем заданное значение x.
f(x) = x³ − 4x → f′(x) = 3x² − 4 → f′(2) = 12 − 4 = 8

Если нужно найти максимум или минимум, ищут точки, где f′(x) = 0, а затем проверяют, как меняется знак производной.

В прямоугольном треугольнике: sin — противолежащий катет / гипотенуза, cos — прилежащий катет / гипотенуза, tg = sin/cos.

Степени, корни и логарифмы

aᵐ·aⁿ = aᵐ⁺ⁿ   ·   aᵐ/aⁿ = aᵐ⁻ⁿ   ·   (aᵐ)ⁿ = aᵐⁿ

Корень √a — неотрицательное число, квадрат которого равен a. Для √A = b нужно b ≥ 0; затем можно возвести обе части в квадрат.

Логарифм logₐb — степень, в которую надо возвести a, чтобы получить b.

log₂8 = 3, потому что 2³ = 8

ОДЗ логарифма: выражение под логарифмом больше нуля. В неравенствах при основании a > 1 знак сохраняется, а при 0 < a < 1 — меняется.

Функции, вершина и системы

В выражении y = c − (x − a)² квадрат всегда неотрицателен, поэтому максимальное значение равно c и достигается при x = a. Для y = (x − a)² + c аналогично минимум равен c.

y = 5 − (x − 2)² ≤ 5 → максимум 5 при x = 2

Систему двух линейных уравнений удобно решать сложением: сложите или вычтите уравнения так, чтобы одна переменная исчезла. Затем найдите вторую подстановкой.

Геометрия

Прямоугольный треугольник: c² = a² + b²

Круг: длина C = 2πr, площадь S = πr², диаметр d = 2r.

Прямоугольный параллелепипед: объём V = abc, пространственная диагональ d = √(a² + b² + c²).

Рисуйте маленький чертёж и подписывайте данные — так проще увидеть нужную формулу.

Проценты и вероятность

Чтобы уменьшить число на p%, умножьте его на (1 − p/100). Чтобы увеличить — на (1 + p/100).

Скидка 15%: 2400 · 0,85 = 2040

При равновозможных исходах вероятность равна:

P = число подходящих исходов / число всех исходов

Ответ вероятности всегда находится от 0 до 1.
